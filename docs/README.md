# Multi-Group Maximum Entropy Method

The MGME method is a coarse-grained model for modelling translational non-equilibrium by evolving moments of the Boltzmann equation. The moment equations for the conserved properties, mass, momentum, and energy, are obtained by multiplying the Boltzmann equation by velocity polynomials and integrating. Closure to the set of moment equations is achieved by assuming each velocity space group is at maximum entropy, or equilibrium.

The code is suitable for applications where translational non-equilibrium is important such as hypersonic and rarefied flows as well as plasmas.

## Installation

The code is written in Python.
